# beatport

Beatport — electronic music store, downloads, and streaming for DJs. Operates a v4 OAuth REST API at api.beatport.com (partner-gated, Swagger-UI portal at /v4/docs/); no public OpenAPI download, no official SDK or CLI. Beatport Group also covers Beatsource, DJCITY, BeatportAmp, Plugin Boutique, Loopcloud, Label Radar, Beatport Live, Beatportal, and IMS.
